# Page Logos

Drop branded logo images in this folder and the dashboard will use them
automatically in each page's header tile. Filenames are matched to page
icon keys:

| File              | Page               |
|-------------------|--------------------|
| `grid.png`        | Overview           |
| `trophy.png`      | Leaderboard        |
| `pulse.png`       | Active Positions   |
| `trend-up.png`    | Performance        |
| `list.png`        | History            |
| `hex.png`         | Coin Analysis      |
| `brain.png`       | Lookback Insights  |

Accepted extensions: `.png` (recommended), `.svg`, `.webp`, `.jpg`.

## Recommended specs

- **Size:** 512×512 px (renders at 44×44 with crisp retina scaling)
- **Background:** transparent
- **Style:** flat, geometric, metallic/holographic, neon accents
  (cyan `#38d0ff` and green `#00e389`)
- **No text, no borders** — the dashboard renders its own tile around it.
